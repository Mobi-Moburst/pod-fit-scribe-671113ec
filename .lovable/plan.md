

## EMV Enhancement Plan

### Context
At $2,500/month ($7,500/quarter), the EMV number needs to consistently exceed that threshold so clients see clear ROI. Currently EMV is hardcoded at $50 CPM with a 40% speaking time assumption, and there's no explanation visible to campaign managers.

### Changes

#### 1. Add methodology tooltips to all Additional Value Metric KPI cards
Add `tooltip` props to each KPICard in the Additional Value Metrics section of `Reports.tsx`:

- **EMV**: "Based on audience size × industry CPM rate × guest speaking time. Reflects the equivalent cost to reach this audience through paid podcast advertising."
- **Social Value**: "Calculated from follower reach across LinkedIn, Meta, YouTube, TikTok, and X using platform-specific ad rates with visibility and premium content multipliers."
- **SOV / GEO / Content Gap**: Similar concise methodology descriptions.

The `KPICard` component already supports a `tooltip` prop with an info icon — just need to pass the strings.

#### 2. Make CPM and speaking time percentage configurable
- Add a small settings section in the report generation form (collapsible "Advanced Settings") with:
  - **CPM Rate** input (default: $50, range $25–$150)
  - **Speaking Time %** input (default: 40%, range 20–60%)
- Store these values in component state and pass them through to `generateReportFromMultipleCSVs` / `generateMultiSpeakerReport`
- The `cpm` param already flows through the pipeline; just need to stop hardcoding `50`
- Add `speaking_time_pct` as a new parameter alongside `cpm` in `calculateEMV()` (currently hardcoded `0.40`)
- Save both values in the report JSON (CPM already saved as `reportData.cpm`, add `speaking_time_pct`)

#### 3. Creative boost: Add a "Total Campaign Value" rollup metric
Combine EMV + Social Value into a single headline number displayed as a new KPI card called **"Total Campaign Value"**. This aggregates all earned media into one impressive figure that's always well above the $7,500 quarterly spend. It's defensible because it's just the sum of two independently calculated metrics.

- Add to the top of the Additional Value Metrics section
- Shows combined EMV + Social Value
- Tooltip: "Combined earned media value and social amplification value generated by this campaign."
- Only visible when both EMV and Social Value have data

### Files to modify
- `src/pages/Reports.tsx` — Add tooltips to KPI cards, add Advanced Settings inputs for CPM/speaking%, add Total Campaign Value card, pass configurable values to generators
- `src/utils/reportGenerator.ts` — Accept `speaking_time_pct` parameter in `calculateEMV()`, `applyEMVCalculations()`, and top-level generator functions; store in report data
- `src/types/reports.ts` — Add `speaking_time_pct` field to `ReportData`
- `src/components/reports/EMVAnalysisDialog.tsx` — Show the CPM and speaking time % used in the methodology section
